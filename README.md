## 📖 Project Overview
**President University Phishing Detector for Students and Lecturers (PUPDfSL)** is a proactive, AI-driven cybersecurity SaaS (Software as a Service) initiative. In an era where social engineering and phishing attacks are increasingly sophisticated, PUPDfSL serves as a critical frontline defense. 

Engineered specifically to safeguard the digital identities, research data, and communications of the President University ecosystem, this system bridges the gap between theoretical machine learning and practical, everyday cybersecurity.

## ✨ Core Features & Capabilities
* 🧠 **Grade V AI-Advanced Architecture:** Powered by a robust Random Forest Ensemble (200 estimators) and Tri-gram NLP extraction, achieving an optimal **99.85% detection accuracy**.
* 🎯 **Target Vulnerability Profiler:** An Early Warning System that cross-references target email accounts with local telemetry to detect recurring attacks and breached identities.
* 🚦 **Heuristic Risk Scoring:** Dynamically classifies payloads into Low, Medium, High, or Critical threat levels based on extracted malicious vectors (e.g., urgency triggers, credential requests).
* 📊 **Threat Intelligence Center:** An interactive, Plotly-powered telemetry dashboard serving as a localized Security Operations Center (SOC) to monitor real-time attack distributions.
* 📩 **Automated SOC Reporting:** Dispatches comprehensive, beautifully formatted threat intelligence reports directly to the user's inbox via the SendGrid API.
* ⚙️ **Dual-Core Fail-Safe System:** A fault-tolerant architecture seamlessly bridging a high-speed FastAPI Backend with a mobile-first Streamlit Frontend.

## 🛠️ Technology Stack
* **Frontend:** Streamlit (with Custom Fluid CSS & Adaptive Mobile UI)
* **Backend API:** FastAPI & Uvicorn
* **Machine Learning Engine:** Scikit-Learn (Random Forest Classifier, TfidfVectorizer)
* **Data Visualization:** Plotly Express & Graph Objects
* **Database & Storage:** MySQL (Local Logging) & Joblib (Model Serialization)
* **External Integrations:** SendGrid API
